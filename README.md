# Проект «Чистота» (Vue + Vite)

Учебный проект по макету из Figma. Стек: **Vue 3**, **Vue Router**, **Vite**. #стек - это набор инструментов для создания продукта

## Страницы

- `/` — Главная
- `/about` — О компании
- `/pricing` — Стоимость
- `/register` — Регистрация
- `/login` — Вход
- `/employees` — Список сотрудников
- `/employees/:id` — Карточка сотрудника с отзывами и сертификатом
- `/privacy-policy` — Политика конфиденциальности
- `/faq` — FAQ (часто задаваемые вопросы)

## Как запустить

```bash
npm install
npm run dev
```

Откройте адрес из консоли (по умолчанию `http://localhost:5173`).

## Публикация на бесплатном хостинге GitHub Pages

Проект уже подготовлен для GitHub Pages:

- в `vite.config.js` включён `base: './'` (корректные пути к ассетам в подпапке репозитория);
- роутер переведён в hash-режим (GitHub Pages не поддерживает server-side rewrite для history-mode);
- добавлен GitHub Actions workflow, который автоматически собирает и выкладывает сайт в Pages.

### 1) Загрузите проект в GitHub

1. Создайте новый репозиторий на GitHub (например, `chistota-vue`).
2. Залейте файлы в репозиторий (ветка `main`).

### 2) Включите Pages

1. Откройте репозиторий → **Settings** → **Pages**.
2. В разделе **Build and deployment** выберите **Source: GitHub Actions**.

### 3) Готово

После пуша в `main` workflow **Deploy to GitHub Pages** соберёт проект и опубликует сайт.
Ссылка будет вида: `https://<ваш_логин>.github.io/<имя_репозитория>/`.
